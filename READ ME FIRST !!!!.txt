About Project
Online Product Reservation System is developed using PHP, CSS, and JavaScript. Talking about the project, it contains a user side with an admin panel where a user can view available products and the users can reserve it easily. Additional, the admin plays an important role in the management of this system. In this project, the user has to perform all the main functions from the user side.

About System
Talking about the features of the Online Product Reservation System, the user can simply view all the available products and add to cart. For this, the user has to login through the system. During the checkout procedure, the user can simply buy the product or reserve it. All the reservation activities/records can be viewed from the account section.

About Admin Panel
From the admin panel, the admin has full access to the system. He/she can manage all the products for the system. In order to add a product, the user has to provide product image, name, price, description, model and serial number. Besides, the admin can view a list of registered customer’s account and check his/her reservation activities. It also contains user management for the easy management of the system.

Design of this project is pretty simple so that the user won’t find any difficulties while working on it. Online Product Reservation system in PHP helps the user in reserving online products easily. To run this project you must have installed virtual server i.e XAMPP on your PC (for Windows). How To Run??

Firstly, after starting Apache and MySQL in XAMPP, follow the following steps

1st Step: Extract file
2nd Step: Copy the main project folder
3rd Step: Paste in xampp/htdocs/

Now Connecting Database

4th Step: Open a browser and go to URL “http://localhost/phpmyadmin/”
5th Step: Then, click on the databases tab
6th Step: Create a database naming “ cj_handgunner ” and then click on the import tab
7th Step: Click on browse file and select “cj_handgunner.sql” file which is inside “DATABASE” folder
8th Step: Click on go.

After Creating Database,

9th Step: Open a browser and go to URL “http://localhost/onlineproduct/”

For Admin Panel

Final Step: Go to URL “http://localhost/onlineproduct/handgunner-administrator/”

Provide the username/email id and password which is provided inside the “Readme.txt” file.
